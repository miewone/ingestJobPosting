기업 소개

블록체인 Web 3.0 시장의 게임체인저, 아이오트러스트와 함께 할 동료를 찾습니다.

아이오트러스트는 Web 3.0 시대에 사용자가 더 쉽고, 더 안전하게 블록체인 서비스를 누릴 수 있는 지갑을 만들고자 합니다.
대한민국 블록체인 산업을 선도하고 있는 아이오트러스트는 2018년 D'CENT 지문인증형 지갑을 시작으로 50여 개 이상의 메인넷을 지원하는 D'CENT 소프트웨어 지갑, 언제 어디서든 이용할 수 있는 카드 타입형 지갑을 선보이며 빠르게 성장했습니다.

2022년 6월, 아기유니콘에 선정된 아이오트러스트는 이제 유니콘으로의 도약을 준비하고 있습니다.
언제나 사용자의 편에서 치열하게 고민하고 블록체인 시장의 판도를 바꾸어 나가는 곳.
아이오트러스트에서 Web 3.0 시대를 함께 이끌어갈 동료를 찾고 있습니다.

주요 업무

[직무소개] 이렇게 일을 해요
• Web3.0 기반 웹 프론트엔드 영역을 맡으며 주로 멀티 체인 지갑인 디센트와 디앱(Dapp), 홈페이지, 이벤트 페이지 등의 다양한 서비스를 개발/운영해요.
• 서비스 개발의 기반이 되는 다양한 공통 모듈을 개발하고 컴포넌트를 테스팅/문서화를 하고 있어요.
• 아직은 초기 단계인 Web3.0이 많은 사용자들에게 쉽고, 안전하게 다가갈 수 있도록 더 편리하고 신뢰할 수 있는 서비스들을 개발하고 운영하는 것을 목표로 하고 있어요.
• Web3.0에는 하루가 다르게 많은 변화가 일어나는 만큼, 변화에 민감하고 새로운 것에 늘 호기심을 가지며 팀원들과 협력하여 성장해 나갈 분을 찾고 있어요.
• 프로젝트 회고, 코드 리뷰, 온보딩, 스프린트 등 성장을 위한 다양한 문화가 운영되고 있어요.

[업무소개] 이러한 일을 해요
• 디센트를 사용하는 유저들에게 편리하고 일관된 UX/UI 경험을 제공하고 아토믹 디자인을 적용한 공통 모듈을 전사적으로 개발해요.
• 사용자의 디센트앱 사용 경험을 더 좋게 개선해요.
• ANDROID/IOS 기반으로 하는 Hybrid Mobile App의 웹뷰를 기반으로 서비스를 개발해요.
• 자사 홈페이지 및 임직원들이 서비스를 운영하는데 필요한 어드민 웹 서비스를 개발/운영해요.
• 블록체인 기반 API의 요구사항을 분석하여 서비스에 적용/운영해요.
• 그 외 신규 사업 서비스를 포함한 멋진 프론트엔드 전반 업무를 담당해요.

【기술스택】 이러한 기술을 활용해요
• 환경: Node.js (Vue.js)
• 주요 라이브러리: Vue / Vuex / Webpack / Babel / Web3.js / amplitude
• 도구: Slack / Github / VSCode / Yarn / Storybook

【업무방향】 이러한 경험도 할 수 있어요
• 서비스 전반에 사용되는 공통 컴포넌트를 디자이너와 함께 개발하면서, 컴포넌트 디자인과 더 좋은 아키텍처를 고민하고 개발할 수 있어요.
• 사용자와 가장 가까이에서 소통하는 영역으로 ANDROID/IOS 웹에 안정적으로 적용되는 완성도 높은 컴포넌트를 개발할 수 있도록, 코드 리뷰를 통해 서로의 기술과 지식을 공유하고 개선방안을 함께 고민할 수 있어요.
• 전 세계 수많은 사용자가 사용하는 앱 화면을 개발할 수 있고, 더 나은 사용자 경험을 위해 끊임없이 고민하며 디자이너와 협업을 통해 서비스를 개선할 수 있어요.
• 서비스가 사용자에게 불편함 없이 제공되도록 CI/CD 등 테스트 자동화와 배포를 경험할 수 있어요.
• 스프린트를 통해 다양한 팀원들과 빠르게 서비스를 런칭하고 고도화하면서 서로 성장하는 경험을 할 수 있어요.
• 그 사이 겪는 다양한 문제를 함께 해결하며 많은 지식을 공유해요.

자격 요건

【자격요건】 이러한 분을 찾고 있어요
• 경력, 학력, 연령 전부 무관하고 블록체인 경험도 없어도 괜찮아요.
• 신입 개발자분은 전산학 또는 컴퓨터공학 전공이 필수예요.
• Javascript, Node.js, 비동기 요청 처리가 능숙한 분이 필요해요.
• Webpack을 통한 번들링 과정에 대한 이해도가 있는 분을 찾아요.
• HTML 및 CSS에 대한 이해가 깊은 분이면 좋아요.

우대사항

【우대사항】 이러한 분이면 더욱 좋아요
• Vue.js 사용 경험, Vue2에서 Vue3으로의 마이그레이션 경험, Vue Composition api 기반으로 개발 경험이 있는 분이면 좋아요.
• Webpack4 및 Webpack5 빌드 과정을 상세히 설정할 수 있거나 테스트 코드 작성, JS 번들링 과정 최적화에 능숙한 분이면 더욱 좋아요.
• 블록체인 관련 서비스, 노드와의 API 요청, 블록체인 Contract 작성, Apache 및 NGINX를 이용한 사이트 배포 경험이 있는 분이면 환영해요.
