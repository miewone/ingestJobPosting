기업 소개

"No code All-in-One API Solution, SyncTree"

엔터플은 2013년에 설립되어 국내 최초의 API 관리 솔루션인 싱크트리(SyncTree)를 개발한 IT 스타트업컴퍼니 입니다. 엔터플은 외부 투자업계와 빅테크 기업으로부터 400억 이상의 투자유치를 받았으며, 현재 50명의 구성원이 함께 성장하고 있는 중입니다. 국내 'No-Code API All-in-One 솔루션 싱크트리'의 무한한 질주는 현재도 지속적인 발전을 거듭 중에 있습니다.

싱크트리는 No-Code 기반으로 개발되었으며 다양한 기능 블록들을 Drag&Drop 방식으로 레고 블록을 쌓아 올리듯 손쉽게 API를 만들 수 있으며 전문 SW 개발자들은 물론 약간의 기술지식이 있는 비즈니스 애널리스트들까지도 간단한 학습 후 사용할 수 있는 획기적인 API 통합 솔루션입니다.
SyncTree는 API를 만든 이후 API들을 관리하고 모니터링 할 수 있는 다양한 기능까지 제공합니다. 현재 국내 API 시장 안에서 엔터플은 세계적 기업들과 경쟁하고 있으며 현재 유수의 기업들이 SyncTree를 도입하여 사용 중에 있습니다.

현재 국내 API 시장 안에서 엔터플은 구글(Apigee), IBM(Red Hat), Broadcom(CA technology)과 경쟁하고 있으며 현재 유수의 대기업 에서 싱크트리를 도입하여 사용 중에 있습니다.
싱크트리는 한국본사가 중심이 되어 홍콩지사를 시작으로 아시아권, 북미권으로 확장하여 세계적인 API 관리 솔루션이 되는 것이 엔터플이 꿈꾸는 목표이며 이미 무수히 많은 검증과 놀라운 결과들은 엔터플의 실적으로 증명되고 있는 상황입니다.

현재 엔터플은 국내 유일무이의 독보적인 기술력을 바탕으로 수평적인 업무소통을 통한 자유로운 의견을 제시하며 다양한 시도를 통해 함께 성장하며 만들어 가는 중입니다.

또한 하루 6시간 근무라는 파격적인 복지혜택과 더불어 아낌없는 지원을 통해 구성원들의 잠재력을 이끌어 주며 매일 매일 함께 성장해 나아가고 있습니다.
