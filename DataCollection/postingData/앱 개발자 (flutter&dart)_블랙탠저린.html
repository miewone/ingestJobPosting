기업 소개

안녕하세요, 기술로 사람들의 삶의 1/3을 바꿀,
전세게 10억명 이상 쓸 제품을 만들, 블랙탠저린 입니다.

저희팀은 의류의 생산과 소비방식을 바꿀 팀으로 현재 함께할 멤버를 찾는 중입니다.
관련해 관심 있으시면 아래 내용 참고 부탁드립니다.

우리팀은 기존 현상에 도전 하고 다르게 생각하는 것의 힘을 믿습니다.
기존 현상에 도전하는 우리의 방식 다음과 같습니다.

자신을 찾아가는 과정을 재밌게 만들고,
쉽게 사용할 수 있게 하고, 편리하게 느끼도록 하는 것 입니다.
우리는 이런 생각을 담은 패션 서비스를 만들고 있습니다.

2) 창업대회 1등하고
3) 탑티어 하우스에서 투자 받은 팀은?

주요 업무

당신은 블랙탠저린의 앱 개발자 포지션에 관심이 있나요?
우리도 ‘성장'에 관심있는 당신을 기다리고 있어요.
Flutter를 활용하여 COCON의 Android / iOS Application을 개발을 기반으로,
더 나아가 사용자에게 더 좋은 서비스, 더 좋은 경험을 느끼게 하기 위한 사용자 중심의 서비스를 만듭니다.

• 포지션: 앱 개발자
- Flutter를 이용한 모바일 앱 Android & IOS개발
- React를 이용한 웹뷰 등 프론트 개발 일부
- 자세한 정보 보기: https://black-tangerine.oopy.io/app

앞서 언급드렸듯이 우리는 Flutter를 활용하여 Android / iOS를 동시에 개발하고 있습니다.

폭발적인 생산성을 기반으로 Application을 만들면서,
우리는 우리의 앱을 사용하는 유저에게 더 집중할 수 있게 되었고,
지금 공고를 보고 계신 여러분들도 우리의 멤버가 되신다면,

가장 유저와 맞닿는 영역인 Client에서
더욱 사용하기 쉽고, 효과적이고, 즐거운 앱을 만드는 여정에 함께 하시게 될 것입니다.

우리는 이 여정을 이어가기 위해 아래와 같은 작업을 합니다.

- Flutter를 기반으로 Mobile Application을 개발하고 배포합니다.
- 기획자와 디자이너의 목업을 기반으로 각각의 요소를 Widget화 하여 관리합니다.
- 상태관리 모델은 현재 GetX를 사용하고 있으나
추후 버전에서는 합류하시는 분들의 의견을 취합하여 BloC / Provider로 전환 계획도 있습니다.
- 최대한 재사용 가능한 코드를 만들고 유지하고 활용하는 데에 큰 관심을 두고 있습니다.
- Github과 Notion을 활용한 적극적인 이슈관리를 지향하고 있습니다.
- 개발자에게 있어서 정말 중요한 자기 회고를 적극 권장하고, 이를 회사 내 시스템에 적용 하고 있습니다.

• 지원 자격: 경력직 (서비스 개발 및 운영 경험 필수)
- 앱 개발 경험(안드로이드/iOS/하이브리드 앱 개발 모두 포함)

현재 Client 기술 스택은 기존 앱 개발자들의 의견에 따라가고 있는데요,
좀 더 자세한 기술 스택은 다음과 같습니다.

- Flutter
- State Management
- GetX (현재 적용 중)
- Provider
- BloC
- RiverPod 중 하나로 전환 계획도 있습니다.
- Android
- Kotiln
- iOS
- Swift
- Front Library
- React
- Vue
- Svelte 중 1개 표준을 근 시일 내 확정 예정
- Firebase
- Git / Github / Github Action


자격 요건

- 지원 자격: 경력 우대 (경력은 서비스 개발 및 운영 경험 필수)
- 앱 개발 경험(안드로이드/iOS/하이브리드 앱 개발 모두 포함)
- Flutter 경험이 없어도 괜찮습니다. Flutter를 사용하다 추후 Native로 변경할 계획도 있습니다.
