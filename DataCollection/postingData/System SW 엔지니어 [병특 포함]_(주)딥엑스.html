기업 소개

딥엑스는 인공지능 기술로 인류문명 발전에 기여를 하고자 세계 최고 수준의 AI 반도체 원천기술을 연구개발 하는 스타트업 기업입니다. 딥엑스의 NPU가 적용될 AI 반도체 시장은 2025년까지 85조 이상의 급격한 성장이 예상되는 거대한 시장으로 우리는 빠르게 변하는 시장에서 기술 리더십을 보유하기 위해 원천기술 확보에 매진하고 있습니다. 우리는 이 도전을 함께 할 열정적인 동료를 찾고 있습니다.
AI 기술이 시대의 패러다임으로 자리잡는 길목에서 동료들과 효과적으로 협업하고 의미 있는 성과를 내기 위해 자신의 재능을 적극적으로 활용하실 분들을 딥엑스로 초대합니다.

주요 업무

ㆍ ROM boot code / Boot Loader 개발
ㆍ Core / BSP / Protocol Stack 구현
ㆍ USB, PCle I/F, DDR I/F, SD/eMMC, MIPI I/F, ISP, Codec 등의 드라이버 SW 구현
ㆍ 각종 하드웨어 IP의 초기 테스트 SW 구현
ㆍ PM(Power Management), MMU 등 시스템 모듈 제어 SW 구현
ㆍ 전체 SOC 의 구동 Verification SW 구현과 디버깅

자격 요건

ㆍ 정확한 의사 소통을 통한 근거 기반 업무 추진이 가능한 합리적인 직무 능력
ㆍ 운영체제 SW 설계 및 구현이 가능한 분
ㆍ ASM / C / C++ / Python 언어 코딩의 풍부한 경험
ㆍ ARM / RISC / x86 / x64 Architecture 제어 기술을 갖춘 분
ㆍ 각종 디바이스 드라이버 개발 경험
ㆍ 알고리즘, 네트워크, 컴파일러 실무적 이해
ㆍ JTAG / MTB / ETM / SWD / ICD 등 디버그 툴 사용이 능숙한 분

우대사항

ㆍ NPU / TPU / GPU 등 HW Accelerator 제어 및 최적화 경험이 있는 분
ㆍ Firmware / RTOS / Linux 등 운영체제 개발 경력이 있는 분
ㆍ PMU(Power Management Unit), MMU 등 시스템 모듈 제어 SW 구현이 가능한 분
ㆍ Git, Cofluence, Jenkins, Jira 등 CI/CD 개발 환경 구축 및 운영 경험이 있는 분
ㆍ Linux / Mac / Windows / Android 기반 응용 프로그램 구현 경험이 있는 분
ㆍ Tensorflow / Pytorch / ONNX runtime 등 AI framework / Middleware 이해 및 응용 개발 경험이 있는 분
