기업 이미지

기업 소개

데브시스터즈 QA는,
우리는 데브시스터즈의 모든 게임에 탑재되는 게임 플랫폼을 검증하고, 다양한 API 및 기능의 안정성을 높은 수준으로 유지하기 위해 품질을 관리하는 파트입니다. 여러 팀과의 커뮤니케이션을 통해 효율적인 업데이트 프로세스를 구축할 수 있도록 노력하고, 품질 관리의 효율성을 높이고자 다양한 방법을 활용하고 있습니다. 이를테면 테스트 케이스 작성 및 버그리포팅, 빌드 테스트, 사용성 테스트, 데이터 검증 등이 대표적인 예시입니다. 뛰어나고 열정적인 멤버들과 함께 데브시스터즈의 게임 플랫폼인 DevPlay의 주요 서비스를 발전시키고, 안정성과 신뢰성, 효율성을 높여나갈 QA 전문가를 찾습니다. 담당하고 있는 주요 서비스는 다음과 같습니다.
- 게임 내 멤버십
- 게임 내 인증
- 인앱 결제
- 그 외 다양한 API 기반 서비스들


주요 업무

- 게임 플랫폼 SDK 기능 검수 설계, 로직검수 및 서비스 전반적인 품질관리
- 플랫폼 서비스 기획개선, 업데이트 일정관리
- 테스트 계획 수립, 일정관리, 리스크관리
- QA 프로세스 정의 및 개선 사항 도출, 품질 표준 기준 선정
- 플랫폼 서비스의 품질향상을 위한 테스트 자동화, 서버 성능 테스트 시스템 도입

- 개발 환경에 맞는 QA 프로세스를 수립 및 개선시켜 나가실 수 있는 분
- 직접 QA 프로세스를 구축하고 고도화 해보신 분

우대사항
