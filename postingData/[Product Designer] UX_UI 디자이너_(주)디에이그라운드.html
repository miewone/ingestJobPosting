기업 소개

"No code All-in-One API Solution, SyncTree"

엔터플은 2013년에 설립되어 국내 최초의 API 관리 솔루션인 싱크트리(SyncTree)를 개발한 IT 스타트업컴퍼니 입니다. 엔터플은 외부 투자업계와 빅테크 기업으로부터 400억 이상의 투자유치를 받았으며, 현재 50명의 구성원이 함께 성장하고 있는 중입니다. 국내 'No-Code API All-in-One 솔루션 싱크트리'의 무한한 질주는 현재도 지속적인 발전을 거듭 중에 있습니다.

싱크트리는 No-Code 기반으로 개발되었으며 다양한 기능 블록들을 Drag&Drop 방식으로 레고 블록을 쌓아 올리듯 손쉽게 API를 만들 수 있으며 전문 SW 개발자들은 물론 약간의 기술지식이 있는 비즈니스 애널리스트들까지도 간단한 학습 후 사용할 수 있는 획기적인 API 통합 솔루션입니다. SyncTree는 API를 만든 이후 API들을 관리하고 모니터링 할 수 있는 다양한 기능까지 제공합니다.

현재 국내 API 시장 안에서 유수의 대기업부터 금융, 건설, IT, 커머스, 스타트업까지 다양한 산업군에서 싱크트리를 도입하여 사용 중에 있습니다.

엔터플은 한국본사를 중심으로 홍콩지사를 설립, 아시아 및 북미권으로 SyncTree로 공격적인 마케팅 중에 있으며 세계적인 API 기업들과의 무한 경쟁 속에 이미 무수히 많은 검증과 놀라운 결과들을 실적으로 증명하고 있는 상황입니다.

현재 엔터플은 국내 유일무이의 독보적인 기술력을 바탕으로 수평적인 업무소통을 통한 자유로운 의견을 제시하며 다양한 시도를 통해 함께 성장하며 만들어 가는 중입니다.

엔터플은 기업들이 Back-end에 소비되는 시간과 비용을 줄여주고, 비즈니스 및 부가가치 창출에 집중할 수 있도록 도움을 주는 것이 목표이며, 누구라도 비즈니스에 대한 아이디어만 있으면 온라인상에서 Back-end 걱정 없이 쉽게 구현할 수 있도록 최신 기술을 쉽고 저렴하게 제공하는 것이 바로 우리 엔터플의 미션입니다.

주요 업무

• 요구사항을 이해하고, 필요한 기능이 제품에 반영될 수 있도록 정책을 정의하고 기능을 기획합니다.
• 기능 개발에 필요한 산출물(와이어프레임, 스토리보드등)을 만들고 관리합니다.
• 유관부서(개발, 디자인, QA등)에 산출물에 대한 목표와 의도를 충분히 설명합니다.
• 개발자가 기능 구현 시 구조 설계를 원활히 할 수 있도록 정책과 기능을 정의하고 기획합니다.
• 업무 미팅에 같이 참석하여 개발팀이 제품의 방향, 목표에 대해 명확히 이해할 수 있도록 최대한 노력합니다.
