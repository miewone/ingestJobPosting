기업 소개

AR Technology 플랫폼인 “SQUARS”로 “VIRNECT(버넥트)”의 눈부신 로켓 성장을 재현합니다!
글로벌 XR시장을 개척할 당신을 기다립니다!

SQUARS?
- SQUARS는 AR Technology와 직/간접 연관된 다양한 서비스들을 제공하는 플랫폼 서비스입니다.
- SQUARS는 “Computer Vision , Target Training , Object Targeting , Cloud Recognition” 등의 자체 원천 기술을 플랫폼 서비스로 제공합니다.

SQUARS에서 제공하는 Service
1. SQUARS Studio
- Stand alone & Cooperation 모드의 Web AR/VR Content Authoring 기능을 제공하는 서비스
2. Squars Collaboration 서비스
- 원활한 원격 협업을 지원하기 위해 실시간 Text, Audio, Video Chatting과 Comment, 리소스 공유, 상호 Guide를 주고 받을 수 있는 기능을 제공하는 서비스
3. SQUARS SDK
- Computer Vision, Object Targeting , Object Training , Cloud Recognition를 이용하여 고객이 원하는 Web과 App을 구현 할 수 있도록 제공하는 서비스
4. SQUARS API
- 클라이언트 어플리케이션만 제작하여 바로 서비스 할 수 있도록 다양한 플랫폼 기능을 제공하는 서비스
5. SQUARS Target
- 모바일 디바이스로 웹에서 바로 Object를 타겟팅하고 인식 할 수 있는 기능을 제공하는 서비스
6. SQUARS Open Share Place
- 컨텐츠 저작에 사용되는 다양한 Asset과 소유한 컨텐츠를 공유하는 공간을 제공해주는 서비스

버넥트의 현재: 국내 최고 산업용 XR 솔루션 기업
- 최근 한국가상증강현실산업협회 '코리아 메타버스 어워드' 포함 30건 이상 수상 경력
- XR 단독 분야, 국내 최고/최다 인력 보유(120명+). 90건 이상의 지적재산권
- 2021년, XR 분야에서 시리즈 B 단계 최대 규모 투자 유치(300억)

버넥트의 미래: 글로벌 최고 XR 플랫폼 기업
- 유럽을 포함한 다양한 국가 대상으로 글로벌 진출 준비를 하고 있습니다
(현재 한국 법인은 오스트리아 비엔나에 있는 리서치 센터와 함께 협업을 하고 있습니다.)
- AR Technology 플랫폼인 “SQUARS”를 준비하고 있습니다.
- 코스닥 상장을 계획하고 있습니다.

버넥트 개발센터 소개
버넥트 개발센터는 플랫폼 웹서비스인 SQUARS, 연계된 스탠다드 솔루션 Remote의 기획/디자인/개발을 담당하며
주요 피처에는 아래 내용이 포함됩니다.
- Web AR/VR Content Authoring을 담당하는 SQUARS Studio
- 고객의 App에 직접 탑재하여 Object Targeting , Object Training , Cloud Recognition 등의 기능을 구현하도록 제공하는 SQUARS SDK 서비스
- 고객이 직접 자체 기능을 구현 할 수 있도록 제공하는 SQUARS API 서비스
- 저작에 사용되는 다양한 Asset과 저작된 컨텐츠를 공유 할 수 있는 Squars Open Share Place

복리후생

1. 버넥트만의 주요 혜택
- 합격된 분께는 대기업 IT계열 회사 수준의 연봉을 제공합니다.
- 전직원 스톡옵션을 부여합니다.
- 연간 360만원의 복지포인트를 제공하며, 식비, 교육, 건강검진 등 자유롭게 사용 가능합니다.
- 신규입사자에게는 사이닝 보너스를, 재직자에게는 리텐션 보너스를 제공합니다.
- 추천된 지원자께서 요청된 과제를 수행 하면, 과제 수행 인센티브를 제공합니다.
- 버넥트 유럽 법인(비엔나)으로 순환 근무 기회가 주어집니다. (항공권, 숙소 지원)

2. 유연한 업무 시간과 워라밸을 위해
- 자율 출퇴근제를 운영합니다.
- 월1회 본인이 희망하는 날 조기 퇴근 합니다.
- 야근 시 발생하는 비용(식사, 교통)을 전액 지원하고, 불가피한 초과 근무는 휴가로 보상합니다.
- 장기근속자에게 유급휴가와 휴가지원금을 제공합니다.
- 3년 근속 : 유급휴가 3일, 휴가지원금 200만원
- 5년 근속 : 유급휴가 5일, 휴가지원금 200만원
- 7년 근속 : 유급휴가 7일, 휴가지원금 300만원
- 10년 근속 : 유급휴가 10일, 휴가지원금 500만원

3. 업무에 몰입할 수 있는 환경을 위해
- 신규입사자 분들께 웰컴 키트를 제공합니다.
- 최신 업무 장비를 지원합니다.
- 최신 IT 장비(홀로렌즈2, HMD etc) 활용이 가능합니다.
- 업무효율 향상을 위해 다양한 협업 Tool 을 활용합니다. (Teams, Notion, JIRA, Salesforce etc)
- 스낵바를 운영하며, 다양한 간식과 음료, 커피를 무제한 제공합니다.
- 쾌적한 사무실에서 근무합니다.

4. 소통과 협업을 잘 하기 위해
- 편한 소통과 존중하는 문화를 지향합니다.
- 매주 금요일 전사 세미나를 통해 회사 주요 사항을 공유하고 적극 소통합니다.
- 팀 활동 비용을 지원합니다. (1인당 월 2만원)
- 타 팀과의 소통을 지원합니다. (2주에 한 번, 타 팀과 2시간 식사/티타임 가능)
- 스포츠, 문화 등 동호회 활동을 지원합니다.

5. 역량 개발을 위해
- 국내/해외에서 열리는 산업계 전시회, XR 관련 전시/학회/세미나 참여를 지원합니다.
- 연간 지급되는 복지포인트를 활용한 직무 교육과 도서 구매를 지원합니다.
- 스터디 소모임을 지원합니다.

6. 경조사/명절/기념일을 위해
- 각종 경조사에 휴가, 경조금, 화환, 상조용품 지원합니다.
- 설과 추석에는 선물을 제공하고, 해당 연휴 전날 조기 퇴근합니다.
- 생일을 맞은 직원에게 상품권(10만원)을 지급하고, 당일 조기 퇴근 합니다.
- 회사 창립일에 기념 선물을 제공합니다.
